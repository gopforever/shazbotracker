How to use

  1. Search and Add a card. We cache the full product response (prices & image).
  2. Pick a Condition, optionally set a Price Key override, edit Qty & Unit Cost.
  3. Click Refresh Prices to update cached payloads. Uses proxy if available.
  4. Import/Export your inventory via CSV (or JSON).

Proxy function is netlify/functions/scp.js. Set SCP_TOKEN in Netlify.
